# NexuFed AI Landing Page

This repository contains the source code for the NexuFed AI landing page.

## Tech Stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Local Development

Requirements:

- Node.js and npm (for example via [nvm](https://github.com/nvm-sh/nvm#installing-and-updating))

Steps:

```sh
# 1. Clone the repository
git clone <REPO_GIT_URL>
cd <REPO_DIRECTORY>

# 2. Install dependencies
npm install

# 3. Start the development server
npm run dev
```
